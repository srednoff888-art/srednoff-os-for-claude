description: Create an ExecPlan for a large task.
---

Create an ExecPlan using `.agent/PLANS.md` and `.claude/rules/60-exec-plans.md`.

Include:

- Goal;
- User value;
- Current state;
- Assumptions;
- Questions;
- GitHub research;
- Architecture;
- Steps;
- Security model;
- Testing plan;
- Rollback plan;
- Definition of done.
