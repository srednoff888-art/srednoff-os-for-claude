# 40-quality-gate.md

Перед финальным ответом проверь:

- scope выполнен;
- assumptions записаны;
- GitHub/docs проверены, если нужно;
- код минимален и понятен;
- нет hardcoded secrets;
- ошибки обработаны;
- тесты добавлены/обновлены;
- lint/typecheck/build пройдены, если доступны;
- security риски оценены;
- deploy/rollback описаны, если применимо;
- финальный отчёт содержит команды проверки.
