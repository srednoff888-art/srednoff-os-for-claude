# 60-exec-plans.md

Создавай ExecPlan для больших задач:

- архитектура; БД; auth; payments; deploy;
- несколько модулей; высокий риск; сложная интеграция.

Формат:

```md
# ExecPlan: <name>

## Goal
## User value
## Current state
## Assumptions
## Questions
## GitHub research
## Official docs checked
## Architecture
## Steps
## Data model / migrations
## API contracts
## Security model
## Testing plan
## Rollback plan
## Definition of done
## Progress log
```
